Shell lock
System

Detailed Description

Simple module to provide a password protection for the shell.

@experimental This module is an experimental feature and only shows as a proof of concept how the shell could be protected with a password. Do not expect relevant security from it for production, since Man-in-the-Middle attacks are possible depending on the used connection method!
